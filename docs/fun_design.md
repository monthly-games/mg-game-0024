## 2. 코어 루프 (Core Loop)

```
┌─────────────────────────────────────────────────────────┐
│  1. 페스티벌 이벤트 참가                                 │
│         ↓                                               │
│  2. 크로스오버 미션/레이드 플레이                        │
│         ↓                                               │
│  3. 개인/서버 보상 획득                                  │
│         ↓                                               │
│  4. 다른 게임 연계 보상 수령 ─────────────→ (1로 반복)    │
└─────────────────────────────────────────────────────────┘
```

### 루프 상세 설명

1. **이벤트 참가**: 기간 한정 페스티벌 이벤트 진입
2. **크로스오버 플레이**: 여러 게임 캐릭터/보스가 등장하는 콘텐츠
3. **보상 획득**: 이벤트 전용 보상 + 서버 진행도 보상
4. **연계 보상**: 다른 게임에서 사용 가능한 크로스 보상

## 3. 재미 축 (Fun Pillars)

### 3.1 이벤트/축제 (Festival)
- **목표**: 특별한 시즌의 축제 분위기
- **핵심 경험**: 연말/기념일의 특별한 경험
- **주요 요소**:
  - 테마 데코레이션
  - 특별 BGM
  - 한정 스토리
  - 축제 미니게임

### 3.2 크로스오버 (Crossover)
- **목표**: 여러 게임의 캐릭터가 한자리에 모이는 재미
- **핵심 경험**: 내가 좋아하는 캐릭터들이 다 나온다!
- **주요 요소**:
  - 크로스오버 캐릭터
  - 합동 스토리
  - 콜라보 보스
  - 게스트 등장

### 3.3 단기 집중 (Burst Engagement)
- **목표**: 짧은 기간 강렬한 몰입
- **핵심 경험**: 지금 아니면 못한다는 긴박감
- **주요 요소**:
  - 기간 한정 콘텐츠
  - 일일 미션
  - 진행도 바
  - 카운트다운

## 4. 디자인 레버 (Design Levers)

### 4.1 크로스오버 이벤트 구조
| 레버 | 설명 |
|------|------|
| 캐릭터/보스/규칙 통합 | 여러 게임의 캐릭터/보스/규칙이 한 곳에 모이는 이벤트 구조 설계 |
| 구현 방향 | game_0001 영웅 + game_0003 용병 + game_0006 히어로 합동 파티 |

### 4.2 기간 한정 콘텐츠 묶음
| 레버 | 설명 |
|------|------|
| 레이드/미션/미니게임 패키지 | 기간 한정 레이드/미션/미니게임 묶음 제공 |
| 구현 방향 | 메인 레이드(협동) + 일일 미션 5개 + 미니게임 3종 |

### 4.3 크로스 보상
| 레버 | 설명 |
|------|------|
| 다른 게임 연계 버프/보상 | 다른 게임 플레이/성과에 따라 이 페스티벌에서 버프/보상 제공 |
| 구현 방향 | game_0001 클리어 → 페스티벌 공격력 +10%, 페스티벌 보상 → game_0003 재화 |

## 5. 재미 측정 지표 (KPI)

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| 이벤트 참여율 | 80% 이상 | 이벤트 접속 유저 비율 |
| 일일 이벤트 플레이 | 30분 이상 | 이벤트 콘텐츠 플레이 시간 |
| 크로스 보상 수령률 | 60% 이상 | 연계 보상 수령 유저 비율 |
| 이벤트 완주율 | 50% 이상 | 메인 스토리/레이드 완료 비율 |
| 이벤트 기간 DAU | 평시 대비 200% | 이벤트 기간 일일 활성 유저 |

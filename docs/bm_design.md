## 3. 상세 BM 설계

### 3.1 이벤트 패스 BM

#### 무료 구조
- 무료 트랙으로 핵심 보상 60% 획득 가능
- 이벤트 콘텐츠 무료 입장
- 일일 이벤트 미션 무료

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 프리미엄 패스 | 한정 캐릭터, 스킨, 재화 | $12.99 |
| 프리미엄+ | 즉시 20레벨 + 프리미엄 | $24.99 |
| 포인트 부스터 | 이벤트 포인트 +50% | $4.99 |
| 추가 입장권 | 레이드 추가 입장 | 프리미엄 재화 100 |

### 3.2 한정 크로스오버 BM

#### 무료 구조
- 이벤트 상점에서 조각으로 교환 가능
- 레이드/미션 보상으로 일부 획득
- 크로스 게임 연계 보상

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 크로스오버 뽑기 | 한정 캐릭터 배너 | 프리미엄 재화 3,000 |
| 천장 시스템 | 50연차 한정 캐릭터 확정 | - |
| 크로스오버 스킨 | 타 게임 캐릭터 스킨 | $9.99~$19.99 |
| 올스타 패키지 | 여러 게임 캐릭터 번들 | $29.99~$49.99 |

### 3.3 레이드/미니게임 BM

#### 무료 구조
- 레이드/미니게임 참여 무료
- 일일 보상 지급
- 서버 진행도 보상 무료

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 레이드 패키지 | 레이드 버프 아이템 | $4.99~$9.99 |
| 추가 시도권 | 미니게임 추가 참여 | 프리미엄 재화 50 |
| MVP 패키지 | 기여도 부스트 + 보상 | $6.99 |

### 3.4 크로스 보상 시스템

| 연계 유형 | 내용 | 구조 |
|----------|------|------|
| 플레이 연계 | 타 게임 플레이 → 페스티벌 버프 | 무료 |
| 보상 연계 | 페스티벌 보상 → 타 게임 재화 | 무료/유료 |
| 번들 연계 | 여러 게임 공용 패키지 | $14.99~$49.99 |

### 3.5 시즌 번들 BM

| 번들 | 내용 | 가격대 |
|------|------|--------|
| 페스티벌 패키지 | 재화 + 한정 캐릭터 + 스킨 | $19.99~$49.99 |
| 올스타 번들 | 여러 게임 캐릭터/아이템 | $29.99~$49.99 |
| 얼리버드 패키지 | 이벤트 시작 3일 한정 | $9.99 |

### 3.6 광고 BM

| 광고 유형 | 적용 | 보상 |
|----------|------|------|
| 리워드 광고 | O | 이벤트 포인트, 추가 보상, 레이드 티켓 |

## 6. BM 밸런스 원칙

### P2W 방지선
- 무과금도 핵심 콘텐츠 참여 가능
- 한정 캐릭터 성능은 기존과 유사 수준
- 크로스 보상은 편의성 중심

### FOMO 관리
- 핵심 보상은 무료로 획득 가능
- 한정 캐릭터 복각 시스템 운영
- 페스티벌 부담 최소화

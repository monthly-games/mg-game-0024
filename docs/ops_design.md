## 3. 1년차 라이브 플랜

### 3.1 초기 (출시 시점)

| 주차 | 주요 활동 |
|------|----------|
| 출시 | 첫 페스티벌 경험 제공 |
| 1~2주 | 전체 타이틀 연동 확인 |
| 3~4주 | 첫 페스티벌 마무리 |

**핵심 과제:**
- 크로스 게임 연동 시스템 완성
- 대규모 동시 접속 안정성
- 페스티벌 보상 체계

### 3.2 연간 운영 (페스티벌 중심)

| 페스티벌 | 시기 | 참여 타이틀 |
|----------|------|------------|
| 스프링 페스티벌 | 3~4월 | Year 1 타이틀 중심 |
| 서머 페스티벌 | 7~8월 | 전체 타이틀 |
| 할로윈 스페셜 | 10월 | 선별 타이틀 |
| 윈터 페스티벌 | 12월 | 전체 타이틀 대규모 |

**핵심 과제:**
- 연 3~4회 대형 페스티벌
- 각 페스티벌마다 참여 타이틀 구성
- 한정 콘텐츠/보상 운영

### 3.3 페스티벌 간 운영

| 기간 | 활동 |
|------|------|
| 페스티벌 후 1주 | 보상 정산 + 복귀 이벤트 |
| 페스티벌 간 | 미니 이벤트 + 프리뷰 |
| 페스티벌 전 1주 | 사전 등록 + 티저 |
